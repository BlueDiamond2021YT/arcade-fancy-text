# animate at speed

Causes the text sprite to print character by character at the given speed.

```sig
fancyText.animateAtSpeed(fancyText.create("Hello, World!"), fancyText.TextSpeed.Fast)
```

## Parameters

* **speed**: the speed at which the text should print
